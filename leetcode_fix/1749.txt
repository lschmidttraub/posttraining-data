You are given an integer array nums. The absolute sum of a subarray [numsl, numsl+1, ..., numsr-1, numsr] is abs(numsl + numsl+1 + ... + numsr-1 + numsr).
Return the maximum absolute sum of any (possibly empty) subarray of nums.
Note that abs(x) is defined as follows:

If x is a negative integer, then abs(x) = -x.
If x is a non-negative integer, then abs(x) = x.

 
Example 1:

Input: nums = [1,-3,2,3,-4]
Output: 5
Explanation: The subarray [2,3] has absolute sum = abs(2+3) = abs(5) = 5.

Example 2:

Input: nums = [2,-5,1,-4,3,-2]
Output: 8
Explanation: The subarray [-5,1,-4] has absolute sum = abs(-5+1-4) = abs(-8) = 8.

 
Constraints:

1 <= nums.length <= 105
-104 <= nums[i] <= 104

To solve the problem of finding the maximum absolute sum of any subarray in the given array `nums`, we can use a variation of Kadane's algorithm. The algorithm will help us find the maximum subarray sum and the minimum subarray sum separately. The absolute sum of any subarray will be the maximum of the absolute values of these two sums.

Here's the step-by-step approach:

1. Initialize variables to keep track of the maximum subarray sum (`max_sum`), the minimum subarray sum (`min_sum`), the current subarray sum (`current_sum`), and the result (`result`).
2. Iterate through the array, updating the `current_sum` by adding the current element.
3. Update the `max_sum` to be the maximum of itself and the `current_sum`.
4. Update the `min_sum` to be the minimum of itself and the `current_sum`.
5. If `current_sum` becomes negative, reset it to 0 because a negative sum would not contribute to a maximum sum in the subsequent subarray.
6. Similarly, if `current_sum` becomes positive, reset it to 0 if we are looking for the minimum sum.
7. The result will be the maximum of the absolute values of `max_sum` and `min_sum`.

Here is the implementation:

```python
from typing import List

class Solution:
    def maxAbsoluteSum(self, nums: List[int]) -> int:
        max_sum = 0
        min_sum = 0
        current_sum = 0
        result = 0
        
        for num in nums:
            current_sum += num
            max_sum = max(max_sum, current_sum)
            min_sum = min(min_sum, current_sum)
            # Reset current_sum to zero if it becomes negative for max_sum calculation
            if current_sum < 0:
                current_sum = 0
        
        current_sum = 0
        
        for num in nums:
            current_sum += num
            # Reset current_sum to zero if it becomes positive for min_sum calculation
            if current_sum > 0:
                current_sum = 0
            min_sum = min(min_sum, current_sum)
        
        result = max(abs(max_sum), abs(min_sum))
        return result
```

However, we can optimize the solution to a single pass by combining the max and min sum calculations in one loop:

```python
from typing import List

class Solution:
    def maxAbsoluteSum(self, nums: List[int]) -> int:
        max_sum = 0
        min_sum = 0
        current_sum = 0
        result = 0
        
        for num in nums:
            current_sum += num
            max_sum = max(max_sum, current_sum)
            min_sum = min(min_sum, current_sum)
        
        result = max(abs(max_sum), abs(min_sum))
        return result
```

This solution efficiently computes the maximum absolute sum of any subarray in a single pass through the array, making it suitable for large input sizes as specified in the constraints.
